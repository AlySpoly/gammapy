*******************************************
Proposals for improvement of Gammapy (PIGs)
*******************************************

Proposals for improvement of Gammapy (PIGs) are short documents proposing a major
addition or change to Gammapy. See PIG 1 for further information.

Below is a list of merged PIGs, i.e. the ones that are finalised, either as accepted or resolved.
The ones that are under discussion can be found on Github as `pull requests with the "pig" label`_.
